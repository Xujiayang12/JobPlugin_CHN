# VOCALOID插件介绍
## 什么是JobPlugin
Job Plugin 提供了一种渠道，可以根据用户的需要，通过编写Lua语言脚本扩展VOCALOID编辑器的功能。这个机制是由Job Plugin API提供的，它提供了获取音乐选区参数信息和更改参数的功能。本文的API版本为3.0.1.0
## Job Plugin API 功能概述
1. 检索和更新音符的基本属性
2. 检索和更新音符的控制参数
3. 检索和更新音符的颤音类型
4. 检索和更新音符的颤音长度
5. 检索和更新音乐选区的属性
6. 获取主音轨的BPM
7. 获取主音轨的时间特征
8. 获取序列的属性
9. 获取和WAV片段的属性
